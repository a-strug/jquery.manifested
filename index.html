Documentation

Options

manifested
A DOM element which will receive the manifested event. Possible values:
'selector', $('selector'), document.getElementById('elementId')
Default: false

This option is required

direction
This options set the direction of manifestation. Possible values: 'TB', 'LR'. Default: 'TB'

'TB' - from Top to Bottom, 'LR' - from Left to Right

speed
Is the float value of milliseconds determining how long the animation will run. Default: 1000
timer
Is the float value of milliseconds that the animation will run. Default: 0
color
Is the string, HEX or RGB value for the color, which will be filled a manifested region.
Possible values: 'red', '#fff', 'rgb(0, 0, 0)'.
Default: #000
height
Is the float value of pixels for height or width (see note) for manifested region. Default: 100

This option set the height for direction 'TB' or width for 'LR'
